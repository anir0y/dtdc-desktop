# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-07

### Added
- Initial release of DTDC Desktop Tracker
- Real-time DTDC shipment tracking
- Visual milestone progress indicator
- Detailed timeline with Load More pagination
- Recent searches feature (shows last 5 tracking IDs)
- Clickable recent searches for quick access
- Home button for easy navigation back to search
- Active tracking indicator in recent searches
- Beautiful DTDC-branded UI with red/orange gradients
- Dark mode support (auto-detects system preference)
- Tab navigation (Tracker and About)
- About page with author information and features
- Local logging to `dtdc_log.jsonl`
- Responsive design for all screen sizes
- Cross-platform support (macOS, Windows, Linux)
- GitHub Actions workflow for automated builds
